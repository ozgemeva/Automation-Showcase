# Automation Showcase
API test automation project built from scratch to learn and demonstrate clean API testing practices.

## Scope
- API test automation with REST Assured
- Request and response validation
- Basic positive and negative test scenarios

## Out of scope
- UI automation
- Mobile automation
- Performance testing
- BDD / Cucumber

## Tech Stack
- Java
- REST Assured
- TestNG
- Maven

## Goal
To learn how API tests are designed, written, and validated in real-world projects.
